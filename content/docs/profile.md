## Keahlian Inti & Spesialisasi

### 1. Cloud Orchestration & Infrastructure as Code (IaC)

* **Automated Provisioning:** Mahir menggunakan **Terraform** untuk mendefinisikan dan membangun infrastruktur cloud secara deklaratif, memastikan replikasi lingkungan yang cepat dan konsisten.
* **Configuration Management:** Mengimplementasikan **Ansible** untuk otomatisasi konfigurasi server, manajemen paket, dan pengamanan sistem (*system hardening*) tanpa agen (*agentless*).
* **Hybrid Cloud Management:** Berpengalaman dalam mengelola beban kerja pada **Google Cloud Platform (GCP)** serta optimasi distribusi Linux Enterprise (Ubuntu, CentOS, Debian).
* **Containerization:** Transformasi arsitektur tradisional ke layanan berbasis *container* menggunakan **Docker** untuk meningkatkan portabilitas dan efisiensi sumber daya.

### 2. Network Engineering & Security Hardening

* **Enterprise Networking:** Merancang topologi jaringan skala besar menggunakan perangkat **MikroTik**, mencakup optimasi *Routing*, segmentasi VLAN, dan konektivitas aman melalui VPN *Site-to-Site*.
* **Edge Delivery & Security:** Konfigurasi **Nginx** dan **HAProxy** sebagai *load balancer* dan *reverse proxy*, serta penerapan kebijakan *firewall* ketat dan enkripsi SSL/TLS untuk menjaga integritas data.

### 3. DevOps Lifecycle & Reliability

* **CI/CD Pipelines:** Membangun alur integrasi dan pengiriman berkelanjutan (**Continuous Integration/Continuous Deployment**) untuk mengotomatisasi pengujian dan peluncuran aplikasi, meminimalkan *human error* dan mempercepat *time-to-market*.
* **Full-Stack Observability:** Mengembangkan sistem pemantauan proaktif menggunakan **Prometheus** dan **Grafana** untuk visualisasi metrik *real-time*.
* **Incident Management:** Memanfaatkan sistem peringatan dini (*automated alerting*) untuk mempercepat *troubleshooting* dan menjaga ketaatan terhadap *Service Level Agreement* (SLA).

### 4. Data Center Operations & Physical Infrastructure

* **Hardware Lifecycle:** Spesialis dalam perakitan, instalasi, dan pemeliharaan preventif infrastruktur rak server serta perangkat penyimpanan data.
* **Capacity Planning:** Melakukan analisis beban kerja untuk perencanaan ekspansi sumber daya fisik guna mendukung pertumbuhan bisnis jangka panjang.
