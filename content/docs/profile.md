# Wildan Firmani Quraisi

IT Infrastructure Engineer yang berfokus pada keandalan sistem (system reliability), keamanan jaringan, dan efisiensi operasional. Saya mengintegrasikan infrastruktur fisik (bare-metal) dengan teknologi cloud modern untuk menciptakan solusi yang tangguh, skalabel, dan efisien secara biaya.

## Infrastructure Modernization & Cloud Management

Infrastructure as Code (IaC): Mengotomatisasi proses provisioning infrastruktur secara deklaratif menggunakan Terraform dan menjalankan manajemen konfigurasi agentless dengan Ansible.

Cloud & OS Administration: Spesialis dalam pengelolaan beban kerja pada Google Cloud Platform (GCP) serta administrasi sistem operasi Linux Enterprise (Ubuntu, Debian, CentOS).

Containerization: Memimpin transformasi dari arsitektur tradisional ke layanan berbasis kontainer yang efisien menggunakan Docker.

## Network Engineering & Security Hardening

Enterprise Networking: Merancang topologi jaringan skala besar menggunakan ekosistem MikroTik, mencakup segmentasi VLAN, routing dinamis (OSPF, BGP), dan konektivitas terenkripsi melalui VPN Site-to-Site.

Traffic Management: Mengoptimalkan distribusi lalu lintas melalui implementasi Nginx dan HAProxy sebagai Load Balancer sekaligus Reverse Proxy.

Security: Menerapkan kebijakan firewall yang ketat dan standar pengamanan perimeter jaringan untuk menjaga kerahasiaan serta integritas data.

## DevOps Lifecycle & Observability

CI/CD Automation: Membangun alur otomatisasi delivery aplikasi yang cepat guna meminimalkan kesalahan manusia (human error) dalam siklus peluncuran produk.

Full-Stack Monitoring: Mengimplementasikan pemantauan proaktif dan sistem peringatan dini secara real-time menggunakan Prometheus dan Grafana untuk memastikan kepatuhan terhadap target SLA.

## Physical Data Center Operations

Hardware Lifecycle: Menangani instalasi fisik, pemeliharaan preventif rack server, hingga penyelesaian masalah (troubleshooting) pada level perangkat keras.

Capacity Planning: Melakukan analisis beban kerja untuk memproyeksikan kebutuhan sumber daya secara presisi dan merencanakan ekspansi infrastruktur fisik perusahaan.

## Research, Development & End-to-End Optimization

Untuk memperkuat pemahaman infrastruktur dari sisi aplikasi, saya secara aktif mengembangkan perangkat lunak lintas platform menggunakan Flutter (Dart) dan layanan backend berkinerja tinggi dengan Go (Golang). Pendekatan end-to-end ini memungkinkan saya untuk mengoptimalkan konfigurasi server berdasarkan karakteristik nyata dari lalu lintas aplikasi di sisi pengguna akhir.
